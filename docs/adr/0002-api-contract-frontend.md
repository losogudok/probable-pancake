# 0002 — Фронтенд приведён к API.md; заморозка разделов без эндпоинтов

Фронтендовый слой `KnowledgeApi` и моки переписаны под точный контрак `API.md` (§1–§9: `/api/search`, `/api/filters/options`, `/api/literature-review`, `/api/graph/subgraph`, `/api/contradictions`, `/api/dashboard/*`, `/api/export/{fmt}`). Роль передаётся в теле `/api/search` (поле `role`); `external_partner` отсекается по `sensitivity` с `hidden_count`. Разделы «Граф знаний» (entity-виз с автокомплитом) и «Источники/Импорт» оставлены файлами под `// @ts-nocheck` и убраны из `router.tsx`/навигации: в `API.md` нет эндпоинтов entity-поиска, библиотеки документов и загрузки файлов — `/api/graph/subgraph` требует `doc_ids`, а не `entityId`. UI сохранён для повторного подключения при появлении соответствующих эндпоинтов в будущих версиях контракта. Экспорт переключён с клиентского генератора (pdfmake) на `POST /api/export/{markdown|jsonld|pdf}`, мок синтезирует blob из `SearchResponse`.
