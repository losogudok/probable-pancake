# React + Vite для пользовательского интерфейса

Основной frontend реализуется на React, Vite и TypeScript вместо Streamlit: продукту нужны сложное клиентское состояние, интерактивный граф, полноценный мобильный UX, компонентное тестирование и прямой путь к production. FastAPI остаётся backend-контрактом; до его стабилизации mock- и HTTP-адаптеры реализуют единый интерфейс `KnowledgeApi`.
