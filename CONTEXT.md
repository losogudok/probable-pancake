# CONTEXT — Научный клубок

Глоссарий предметной области. Без имплементационных деталей.

## Сущности (canonical)

- **Material** — вещество/материал (никель, медь, гипс, сульфаты, хлориды, техногенный гипс, угольные отходы).
- **Process** — технологический процесс (выщелачивание, электроэкстракция, обессоливание, плавка).
- **Equipment** — оборудование/установка (ванны электроэкстракции, печь взвешенной плавки/ПВП, системы очистки газов).
- **Facility** — площадка/фабрика/лаборатория (обогатительная фабрика, рудник).
- **Experiment** — опыт/протокол: материал + режим + результат.
- **Publication** — статья/обзор/доклад/диссертация.
- **Patent** — патентный документ.
- **Expert** — носитель компетенции (автор, исследователь, лаборатория).
- **Property / Parameter / Condition** — свойство (выход металла), параметр (концентрация, температура, скорость потока), условие (холодный климат, производительность).
- **Recommendation** — подтверждённый вывод/область применения.

## Связи (canonical глаголы)

- `uses_material` / `uses_reagent` / `uses_equipment` / `uses_method`
- `performed_at` / `performed_by` / `validated_by` / `contradicted_by`
- `requires` / `has_condition` / `has_parameter` / `has_result`
- `derived_from` / `cites`

## Дисциплины (домены)

- **Гидрометаллургия** — выщелачивание, электроэкстракция, обессоливание.
- **Пирометаллургия** — плавка (ПВП), штейн/шлак.
- **Экология** — очистка газов (SO₂), шахтные воды.
- **Переработка отходов** — техногенный гипс, угольные отходы.

## География (canonical)

- **Отечественная практика** — Россия/СНГ.
- **Зарубежная практика** — вне РФ.
- **Глобальная** — общеприменимые/нерегиональные.

## Верификация

- **Источник** — origin публикации/отчёта.
- **Уровень достоверности** — шкала 1–5 звёзд по типу источника.
- **Дата актуализации** — когда факт внесён/обновлён в графе.
- **Противоречие** — числовое расхождение по одному параметру >30% между источниками.
- **Пробел** — комбинация материал×процесс×условие без экспериментов/публикаций.

## Термины-синонимы (RU/EN)

- электроэкстракция = electrowinning
- ПВП = печь взвешенной плавки = fluidized bed furnace
- штейн = matte; шлак = slag
- МПГ = металлы платиновой группы = PGM
- кучное выщелачивание = heap leaching
